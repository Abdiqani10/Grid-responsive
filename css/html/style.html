About Me

Tim Berners-Lee is a British computer scientist credited with inventing the World Wide Web (WWW). Berners-Lee enabled a system to be able to view web pages (hypertext documents) through the internet. He also serves as a director for the World Wide Web Consortium (W3C) which oversees standards for the Internet and World Wide Web. Today he speaks on building industries on the Web, on electronic commerce, creativity in cyberspace and on the future of the Internet.
